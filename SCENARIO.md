# HoneyComb 게임 시나리오 가이드 (Scenario Design Doc)

이 문서는 **게임 시나리오·플로우·연출** 기준입니다. 스토리, UI 문구, 이벤트 순서, 튜토리얼 등을 바꿀 때 여기를 참고하고, 변경 후 문서도 갱신하세요.

---

## 1. 현재 시나리오 요약

| 단계 | 설명 | 트리거 |
|------|------|--------|
| **시작** | 레벨 1, 다이아 0/N, 타이머 시작 | 페이지 로드 또는 세션 복구 |
| **플레이** | 블록 스왑 → 3개 이상 매치 → 제거 → 다이아 수집 | 사용자 입력 |
| **콤보** | 리필 후 연속 매치 시 연쇄(더블/트리플/익사이팅) 연출 | 매치 후 자동 리필 + 추가 매치 |
| **레벨업** | 필요 다이아 달성 시 다음 레벨, 보드·시간 리셋 | 다이아 목표 달성 |
| **게임 오버** | 시간 만료 시 결과 화면 → 다시 하기 | 타이머 0초 |
| **새 게임** | 레벨 1부터 재시작, 세션 삭제 | 상단 "새 게임" 버튼 |

---

## 2. 시나리오 관련 요소 & 코드 위치

### 2.1 진입·안내 문구 (HTML)

| 위치 | 파일 | 요소/클래스 | 내용 |
|------|------|-------------|------|
| 타이틀 | HoneyComb.html | `<h1>` | HoneyComb |
| 서브 | HoneyComb.html | `.sub` | 클릭↔클릭으로 인접 블록 맞바꾸기 · 3개 이상 맞으면 제거 |
| 조작 힌트 | HoneyComb.html | `.mode-hint` | 클릭→클릭: 인접 블록 맞바꾸기 (3개 이상 맞으면 제거) |
| 콤보 힌트 | HoneyComb.html | `.combo-hint` | 리필 후 또 맞으면 **연쇄** → 더블·트리플·익사이팅 |
| 새 게임 버튼 | HoneyComb.html | `#newGameBtn` | 새 게임 |

### 2.2 인게임 연출 문구 (JS)

| 연출 | 파일 | 함수/변수 | 내용 예시 |
|------|------|-----------|-----------|
| 콤보 라벨 | honeycomb-game.js | `getComboLabel(count)` | 2연쇄 콤보! (더블), 3연쇄 콤보! (트리플), N연쇄 콤보! (익사이팅) |
| 콤보 배지 | honeycomb-game.js | `G.badgeText` | x2!, x3!, AMAZING! |
| 대량 제거 문구 | honeycomb-game.js | `G.bigText` | 대박!, 🎉 대박!! (6개 이상/8개 이상) |
| 레벨업 문구 | honeycomb-draw.js | levelUpUntil 블록 내 | ⭐ 레벨 N ⭐ |
| 게임 오버 문구 | honeycomb-draw.js | `G.gameOver` 블록 | ⏱ 시간 초과!, 최종 점수: N, 아래 버튼을 눌러 주세요 |
| 다시 하기 버튼 | HoneyComb.html | `#gameOverRestartBtn` | 다시 하기 |

### 2.3 플로우 제어 (JS)

| 흐름 | 파일 | 관련 변수/함수 | 설명 |
|------|------|----------------|------|
| 레벨업 대기 | honeycomb-game.js | `pendingLevelUp`, `comboDrawLoop` | 콤보 연출 끝난 뒤 레벨업 처리 |
| 레벨업 연출 길이 | honeycomb-state.js | `LEVEL_UP_DURATION_MS` | 1800ms |
| 게임 오버 표시 | honeycomb-game.js | `gameOver`, `gameOverOverlay` | 타이머 0 + 미션 미달 시 오버레이 표시 |
| 세션 복구 | honeycomb-session.js | `loadSession` | 24시간 이내 저장된 레벨/보드/타이머 복구 |

### 2.4 마일스톤·특별 연출 (확장용)

| 구상 | 현재 | 코드 후보 |
|------|------|------------|
| 레벨 10/20 달성 축하 | 없음 | honeycomb-game.js 레벨업 분기, honeycomb-draw.js 문구 |
| 튜토리얼 첫 플레이 | 없음 | honeycomb-main.js 첫 방문 플래그, 별도 안내 레이어 |
| 챕터/에피소드 | 없음 | 레벨 구간별 문구·배경 변경 |
