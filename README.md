# HoneyComb – 벌집 퍼즐

헥스(벌집) 격자 퍼즐 게임입니다.

## 플레이 방법

- **클릭↔클릭**: 인접한 두 블록을 맞바꿔 3개 이상 연속 매치
- **드래그**: 같은 색 블록을 연속으로 연결해 제거 (3개 이상)
- **아이템**: 폭탄(주변 6칸 폭발·연쇄), 미사일(가로 한 줄), 크로스(가로+세로)
- **다이아몬드**: 레벨 클리어에 필요한 개수를 모으세요
- **타이머**: 제한 시간 안에 플레이

## 실행

`HoneyComb.html`을 브라우저에서 열면 됩니다.

## 기술

HTML5 Canvas, Vanilla JavaScript. 반응형·터치 지원.
